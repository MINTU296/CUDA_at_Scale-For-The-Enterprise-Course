# Image Rotation using NVIDIA NPP and CUDA

## Overview

This repository contains a minimal CUDA sample that rotates a grayscale image using NVIDIA Performance Primitives (NPP). It demonstrates how to load an image, perform a GPU-accelerated rotation using `nppiRotate_8u_C1R`, and save the output image.

## Repository layout

- `bin/` - compiled binary output
- `data/` - sample inputs and generated outputs
- `lib/` - optional libraries for linking
- `src/` - source code
- `Makefile` - build configuration
- `README.md` - project documentation
- `run.sh` - runtime helper script

## Prerequisites

- NVIDIA GPU with CUDA support
- CUDA Toolkit installed
- NVIDIA NPP available through CUDA include and library paths
- `nvcc` and a compatible host compiler installed

## Build

From the repository root:

```bash
make
```

If CUDA is installed in a custom location, update the `NVCC`, `CXXFLAGS`, and `LDFLAGS` values in `Makefile`.

## Run

Use the provided make rule:

```bash
make run
```

Or run the binary directly:

```bash
./bin/imageRotationNPP --input data/Lena.png --output data/Lena_rotated.png
```

If no `--input` argument is supplied, the program attempts to resolve `Lena.pgm` from the workspace.

## Output

- Default output filename is generated by appending `_rotate.pgm` to the input basename.
- The current rotation angle is fixed to `45.0` degrees in `src/imageRotationNPP.cpp`.

## Clean

Remove generated binary files:

```bash
make clean
```

## Notes

This sample is intended as a compact example of CUDA/NPP image processing. For production use, extend the project with better command-line validation, larger image support, and more flexible build tooling.
